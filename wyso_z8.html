Zadanie 8 – Wewnętrzny arkusz stylów

  1. Gdzie jest wstawiany wewnętrzny arkusz stylów?
    Wewnętrzny arkusz stylów CSS jest wstawiany w sekcji <head> dokumentu HTML, wewnątrz znacznika <style>.
  2. Składnia teoretyczna (ramka + wcięcia): 

    <head>
  <style>
    selektor {
      cecha: wartość;
    }
  </style>
</head>
  3. Jakie mogą być selektory?
    Selektorami mogą być znaczniki HTML (np. p, h1, div), klasy (.klasa), identyfikatory (#id) oraz selektory uniwersalne (*).
  4. Jakie mogą być cechy i wartości?
    Cecha (właściwość CSS) określa co zmieniamy, np. color, font-size, background-color.
    Wartość określa konkretną ustawienie cechy, np. red, 20px, Arial, blue.
