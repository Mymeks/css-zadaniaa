Zadanie 9 – Styl zewnętrzny

  1. Składnia teoretyczna: 

    <head>
  <link rel="stylesheet" href="style.css">
</head>
  2. Gdzie wstawiany jest styl zewnętrzny, jaki znacznik go obsługuje i ile razy może występować?
    Styl zewnętrzny wstawiany jest w sekcji <head> dokumentu HTML. Odpowiada za to znacznik <link>. W jednym dokumencie można użyć wielu znaczników <link>, aby podłączyć kilka arkuszy CSS.
  3. Najprostsza forma wstawiania stylu zewnętrznego:

    <link rel="stylesheet" href="style.css">
  4. Ile arkuszy stylów można dołączyć i jaka zasada obowiązuje?
    Do jednego dokumentu HTML można dołączyć wiele arkuszy CSS. Obowiązuje zasada kaskadowości – style są nakładane warstwowo.
  5. Co się dzieje w przypadku konfliktów?
    W przypadku konfliktów (te same style w różnych arkuszach), stosowana jest zasada kaskady CSS – pierwszeństwo ma styl o większej ważności, bliżej dokumentu lub zapisany później.
  6. Jak tworzy się i edytuje pliki w edytorze tekstowym?
    Plik CSS tworzy się w zwykłym edytorze tekstowym (np. Notatnik), zapisując go z rozszerzeniem .css i wpisując reguły CSS ręcznie.
  7. Jak tworzy się i edytuje pliki w specjalnym edytorze CSS?
    W edytorach CSS (np. Visual Studio Code, Sublime Text) korzysta się z podświetlania składni, autouzupełniania i podglądu kodu, co ułatwia edycję stylów.
  8. Jakie rozszerzenie ma plik zewnętrznego arkusza stylów?
    Plik zewnętrznego arkusza stylów posiada rozszerzenie .css.
  9. Zalety zewnętrznego arkusza stylów:
    • Ułatwia zarządzanie stylem wielu stron jednocześnie
    • Oddziela treść HTML od wyglądu strony
  10. Uwagi na temat pisania w CSS:
    Należy pamiętać o średnikach po każdej deklaracji, poprawnej składni, oraz o czytelnym formatowaniu kodu dla łatwiejszej edycji.
