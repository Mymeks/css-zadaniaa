CSS – Cascading Style Sheets

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation and appearance of documents written in HTML or XML. CSS allows separation of content from visual design such as colors, fonts, spacing, layout and positioning of elements on a web page.

The abbreviation CSS stands for Cascading Style Sheets. It enables developers to apply styles to multiple web pages at once, making websites easier to maintain, more consistent and visually attractive.
