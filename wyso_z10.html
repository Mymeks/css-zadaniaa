Zadanie 10 – Budowa selektora

  1. Wymień i opisz dwie części selektora:

    1. Selektor – wskazuje element HTML, który będzie formatowany, np. p, h1, div.
    2. Deklaracja – określa styl, który zostanie zastosowany do wybranego elementu.
  2. Znaczenie nawiasów klamrowych { ... }:
    Nawiasy klamrowe { } obejmują blok deklaracji CSS i określają zakres działania stylu dla danego selektora.
  3. Opis cech:
    Cechy (właściwości CSS) określają, co chcemy zmienić w wyglądzie elementu, np. color, font-size, background-color.
  4. Opis wartości:
    Wartości to konkretne ustawienia przypisane do cech, np. red, 20px, Arial, blue.
  5. Kiedy stosujemy średnik (;) i kiedy nie jest konieczny?
    Średnik ; stosujemy do oddzielania kolejnych deklaracji CSS w bloku.
    Nie jest konieczny tylko przy ostatniej deklaracji w bloku, ale zaleca się stosować zawsze dla bezpieczeństwa i czytelności kodu.
