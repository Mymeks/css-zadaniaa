Zadanie 7 – Styl lokalny (liniowy / inline CSS)

  1. Na co pozwala styl lokalny i który element będzie formatowany?
    Styl lokalny (liniowy) pozwala na bezpośrednie formatowanie pojedynczego elementu HTML. Styl zapisuje się w atrybucie style danego znacznika, więc tylko ten konkretny element zostaje sformatowany.
  2. Składnia teoretyczna:

    <znacznik style="cecha: wartość;"> treść </znacznik>
      przykład zapisu CSS wewnątrz elementu HTML

    (znaki < > zostały zapisane jako encje HTML, aby kod się nie wykonywał)
  3. Co może być selektorem?
    Selektorem może być każdy znacznik HTML, np. <p>, <h1>, <div>, <span>, który otrzymuje atrybut style.
  4. Co wpisujemy jako "cechę" i jakie jest znaczenie średników?
    „Cecha” (właściwość CSS) określa, co chcemy zmienić, np. kolor, czcionkę, rozmiar.
    Średnik (;) oddziela kolejne deklaracje CSS i kończy każdą parę cecha: wartość.
  5. Co wpisujemy jako "wartość"?
    Wartość określa konkretne ustawienie danej cechy, np. red, blue, 20px, Arial, bold.
