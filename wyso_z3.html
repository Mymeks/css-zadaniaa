Cechy modyfikowania formatowania w HTML oraz CSS

HTML CSS
Formatowanie bezpośrednio w znacznikach Oddzielenie treści od wyglądu strony
Powtarzanie stylów w wielu miejscach Możliwość użycia jednego stylu dla wielu elementów
Mniejsza elastyczność projektowania Duża kontrola nad wyglądem strony
Trudniejsza modernizacja strony Łatwa zmiana wyglądu całej witryny

Nowe możliwości CSS w porównaniu z HTML

  1. Stosowanie arkuszy stylów dla wielu stron jednocześnie
  2. Precyzyjne pozycjonowanie elementów
  3. Obsługa animacji i efektów wizualnych
  4. Responsywność strony (dopasowanie do urządzeń)
  5. Oddzielenie struktury dokumentu od prezentacji

Wady CSS

  • Różnice w interpretacji stylów przez przeglądarki
  • Możliwość konfliktów między stylami
  • Wymaga dodatkowej nauki składni
  • Błędy w CSS mogą wpływać na wygląd całej strony
